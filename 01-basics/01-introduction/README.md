# Introduction to Node.js

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It
allows you to run JavaScript on the server-side, enabling the development of
scalable and high-performance applications.

## Key Features

- **Event-driven architecture**: Non-blocking I/O model for efficient and
  lightweight processing.
- **Single-threaded**: Uses a single thread for event handling.
- **Asynchronous**: Designed for asynchronous programming with callbacks,
  promises, and async/await.
